### Adoc

> Just another markdown doc editor

PHP + Laravel5.8 + React + bootstrap

#### 安装

```bash
git clone https://github.com/yybawang/adoc.git
cd adoc
cp .env.example .env
// 编辑配置文件中的 APP_URL、数据库、redis连接
vi .env

// 下载包
composer install

// 下载 npm 包，进行前端编译
npm install
npm run prod

// 访问 APP_URL 测试
```
